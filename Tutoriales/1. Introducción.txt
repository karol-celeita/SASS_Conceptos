SASS es un preprocesador, que nos permite escribir código CSS de una manera más dinámica,
agregando sintaxis de un lenguaje de programacion, 
incluyendo variables. funciones, módulos y distintos tipos de valores

¿Por que SASS?
- Totalmente compatible con CSS
- Codigo mas rapido
- Muchos framework se construyen en SASS
- Hay mucha documentacióm
